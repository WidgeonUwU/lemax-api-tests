# lemax-api-tests
Postman API тесты для внутреннего учебно-коммерческого проекта от компании LEMAX

## Описание проекта

Проект направлен на тестирование REST API бэкенд-приложения.  
Тесты покрывают основные бизнес-сценарии и ключевые сущности системы, связанные с авторизацией, управлением данными и HR-процессами.

## Область тестирования

В проекте реализованы Postman-коллекции для следующих доменов:
- Авторизация и аутентификация
- Статьи
- Сотрудники
- Аттестации
- Встречи
- Должности
- Профессии

Каждый домен вынесен в отдельную коллекцию для удобства поддержки и запуска.

## Используемые инструменты

- Postman
- JavaScript (тесты в Postman)

## Что проверяется

- Позитивные и негативные сценарии работы API
- Корректность HTTP-статус кодов
- Базовая валидация структуры ответов
- Smoke-проверки критичных эндпоинтов
